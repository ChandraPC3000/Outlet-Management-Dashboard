# NFR Izin Prinsip Dashboard

Dashboard ini dibuat menggunakan **Streamlit** untuk menganalisis data pengajuan izin prinsip tenant Non-Fuel Retail di SPBU Pertamina.

### Fitur:

- Upload file Excel otomatis.
- Filter berdasarkan Region, Kategori, dan Status.
- Visualisasi interaktif menggunakan Plotly.
